DualChatAI is an AI-powered chat system that works exclusively with the **Groq API**.  
Just grab your **Groq API key** from the [Groq Console](https://console.groq.com) and start chatting instantly.

## How It Works
1. Create an account on **[Groq Cloud](https://console.groq.com)**.  
2. Generate your API key from the **Groq Console**.  
3. Copy the key and use it inside DualChatAI to connect with Groq’s LLMs.  
4. Once your key is set, DualChatAI will handle all requests through Groq’s API for fast, high-performance AI chat.
